## API Documentation

This repository contains the client application for The Table for All. Use the guides below to understand and integrate all public components, services, hooks, and configuration.

- Components: see `docs/components.md`
- Services & Hooks: see `docs/services-and-hooks.md`
- Configuration: see `docs/configuration.md`

### Quick start
- Import components from `misszerhungr/client/src/components/*`
- Use the data service and hook from `misszerhungr/client/src/services/FoodInsecurityDataService`
- Configure your Google Maps API key in `misszerhungr/client/src/config/googleMaps.ts`
